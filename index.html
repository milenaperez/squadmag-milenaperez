Cambio magnético, precisión instantánea

Las Smith Squad MAG incorporan el sistema MAG de intercambio magnético, que permite reemplazar los lentes sin perder tiempo ni visibilidad.

  • 9 puntos de anclaje magnético para máxima estabilidad
  • Sistema de bloqueo mecánico para evitar aperturas accidentales

Tecnología, estilo y rendimiento

Semi- Rimless

Lente de gran tamaño que amplía el campo de visión y evita distracciones.

ChromaPop

Tecnología que ajusta automáticamente la transmisión de luz visible.

AirFlow

Ventilación que evita el empañamiento para cambios de velocidad.

Diseñadas para durar, hechas para rendir

Cada detalle está pensado para ofrecerte rendimiento, durabilidad y comodidad todo el día.

  • Lente Carbonic-X
  • Espuma DriWix
  • Correa QuickFit

Lente Carbonic-X

Moldeado individualmente para ofrecer resistencia al impacto, combinado con tratamiento TLT para eliminar distorsiones ópticas. Ventilación activa integrada (Airflow + AirEvac™) que canaliza el aire para reducir el empañamiento

Espuma DriWix

Tres niveles de espuma que ayudan a disipar la humedad, adaptarse al rostro y asegurar comodidad y sellado térmico. La capa base de alta densidad evita puntos calientes, la capa media asegura sellado cómodo, y la capa interna mantiene la piel seca

Correa QuickFit

Correa ultraancha con silicona antideslizante que se ajusta firmemente sobre casco o gorro. Además, mantiene las gafas estables en cada movimiento
